# F1 Race Intelligence

An engineering-focused platform for exploring Formula 1 race data through interactive analytics, visualization, and strategy intelligence.

## Features (Current)

- Historical race ingestion
- Driver comparison
- Interactive dashboards
- Race analytics

## Planned

- Strategy simulator
- Telemetry explorer
- AI race analyst
- Race replay

## Tech Stack

- Python
- FastAPI
- React
- TypeScript
- PostgreSQL
- AWS
- Docker
- GitHub Actions

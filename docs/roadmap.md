# Product Roadmap

This roadmap intentionally favors small, complete milestones over large unfinished features.

---

# Phase 1

## Foundation

Goal

Create the engineering foundation.

Deliverables

- AWS Organization
- GitHub repository
- Development environment
- CI pipeline
- Docker
- Project documentation

---

# Phase 2

## Data Ingestion

Goal

Successfully ingest one Formula 1 session.

Deliverables

- Download race data
- Normalize session information
- Validate data
- Store locally
- Automated ingestion script

Success Criteria

Able to ingest any historical race session.

---

# Phase 3

## Driver Comparison MVP

Goal

Create the first useful product.

Deliverables

- Driver selection
- Lap comparison
- Tire timeline
- Position chart
- Pace summary

Success Criteria

A user can compare two drivers from a race.

---

# Phase 4

## Race Dashboard

Goal

Explore an entire race.

Deliverables

- Race timeline
- Pit stop visualization
- Safety Car events
- Team comparison
- Position evolution

---

# Phase 5

## Telemetry

Goal

Support detailed driver analysis.

Deliverables

- Speed traces
- Brake traces
- Throttle traces
- Gear changes
- Corner comparison

---

# Phase 6

## Strategy Intelligence

Goal

Help explain race strategy.

Deliverables

- Undercut analysis
- Overcut analysis
- Tire degradation
- Pit window analysis
- Race simulations

---

# Phase 7

## AI

Goal

Natural language interface.

Deliverables

- Driver questions
- Race summaries
- Strategy explanations
- Data retrieval
- Guided exploration

AI must use analytics services rather than raw data.

---

# Phase 8

## Public Platform

Goal

Become a polished engineering portfolio.

Deliverables

- Authentication
- Public deployment
- Documentation
- Monitoring
- Testing
- Mobile optimization
- Performance improvements
